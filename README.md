# Vue form with google maps API

## Project setup
```
npm install
```
## Include API Key in main.js

```
Vue.use(VueGoogleMaps, {
  load: {
    key: 'ADD_KEY_HERE',
    libraries: 'places', 
  }
})
```
### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
